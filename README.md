## Project Names
- **API**: `productapi`
- **Frontend**: `productfrontend`
- **Tests**: `productunittest`

## Features
- **Secured API**: .NET 9 Web API with JWT.
- **Frontend**: React (Vite) & Standard CSS/Tailwind
- **Testing**: Unite Testing

## Quick Start

### 1. Backend
```bash
cd productapi
dotnet run
```

### 2. Frontend
```bash
cd productfrontend
npm install
npm run dev
```

### 3. Running Tests
```bash
dotnet test
```
